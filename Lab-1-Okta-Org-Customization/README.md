# 🧪 Lab 1: Customize Your Okta Org

## 🎯 Objective
Customize the branding of an Okta organization by modifying logos, colors, and UI elements.

---

## 🛠️ Steps Performed
- Navigated to Customizations > Brands in Okta Admin Console
- Selected default brand
- Uploaded company logo
- Uploaded favicon
- Changed primary color to #ff9800
- Configured sign-in page settings
- Configured end-user dashboard
- Configured error pages

---

## 🔐 Key IAM Concepts
- Identity branding and user trust
- Customization of authentication interfaces
- Importance of consistent identity experience
